# Frogger Arcade Game

    Frogger Arcade Game is a game to help with focusing, get to the top and avoid the bugs touching you. Every time you win or lose game will start over again

## Installation
    Clone the GitHub repository

    $git clone https://github.com/LissaNgSang/Udacity-FEND.git
    $ cd 4_Intermidiate\ JS/
    $ cd arcade-game
    $ open index.html

## Play Arcade Game
    1. Use arrow keys on keyboard to move around
    2. Avoid bug collision
    3. Go to the top and WIN!
